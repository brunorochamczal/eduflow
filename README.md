🎓 Edunex - Gestão Escolar Inteligente
O Edunex é uma plataforma robusta de gestão escolar desenvolvida para simplificar a administração de instituições de ensino. Unindo uma interface Glassmorphism moderna com um backend eficiente, o sistema oferece controle total sobre matrículas, corpo docente, turmas e saúde pedagógica através de Inteligência de Dados.

🚀 Funcionalidades Principais
🔐 Autenticação Segura: Sistema de login com feedback em tempo real e proteção de rotas.

👨‍🎓 Gestão de Alunos: Matrícula simplificada com organização por série e contato direto com responsáveis.

👨‍🏫 Controle de Docentes: Cadastro de professores e suas respectivas especialidades.

🏫 Gestão de Turmas e Disciplinas: Alocação de salas, turnos e controle de carga horária da grade curricular.

💰 Painel Financeiro: Monitoramento de mensalidades e integração para cobrança via WhatsApp.

🚨 Radar Pedagógico: Algoritmo que analisa frequência e notas para detectar riscos de evasão escolar.

📸 Demonstração do Sistema
1. Tela de Login (Glassmorphism)
Interface moderna com efeito de transparência e desfoque, otimizada para foco total no usuário.

2. Dashboard Principal
Painel centralizado com cards interativos para navegação rápida entre todos os módulos do sistema.

3. Módulos de Cadastro
Formulários dinâmicos e limpos para uma inserção de dados rápida e intuitiva.

🛠️ Tecnologias Utilizadas
Frontend: HTML5, CSS3 (Custom Glassmorphism), JavaScript (Vanilla ES6).

Backend: Python com Flask Framework.

Banco de Dados: PostgreSQL (Hospedado via Neon.tech).

Hospedagem: Railway App.

Ícones: FontAwesome 6+.

⚙️ Instalação e Configuração
Se desejar rodar o projeto localmente:

Clone o repositório:

Bash
git clone https://github.com/seu-usuario/edunex.git
Instale as dependências:

Bash
pip install -r requirements.txt
Configure as Variáveis de Ambiente:
Crie um arquivo .env na raiz e adicione sua conexão:

Snippet de código
DATABASE_URL=sua_string_de_conexao_do_neon
Inicie o servidor:

Bash
python app.py
🎨 Interface e Design
O design foi construído seguindo os princípios de UI/UX Moderno:

Cores: Paleta baseada em Indigo (#6366f1) e Roxo Profundo para um ar educacional e tecnológico.

Responsividade: O layout se adapta para uso em desktops e tablets.

Identidade Visual: Logo centralizada e favicon personalizado para reconhecimento da marca na aba do navegador.
